# تعليمات بناء التطبيق

## المتطلبات الأساسية
1. Node.js مثبت على النظام
2. جميع التبعيات مثبتة

## خطوات البناء

### 1. تثبيت electron-builder
```cmd
npm install --save-dev electron-builder
```

### 2. بناء التطبيق

#### أ. بناء نسخة Installer (مثبت):
```cmd
npm run build-installer
```
سيتم إنشاء ملف Setup قابل للتثبيت في المجلد:
```
D:\AttendanceSystem-Build\
```

#### ب. بناء نسخة Portable (محمولة):
```cmd
npm run build-portable
```
سيتم إنشاء ملف exe محمول لا يحتاج تثبيت

#### ج. بناء كلا النسختين معاً:
```cmd
npm run build
```

## الملفات الناتجة

سيتم إنشاء التطبيق في المجلد:
```
D:\AttendanceSystem-Build\
```

الملفات الناتجة:
- `نظام إدارة الحضور-Setup-1.0.0.exe` - المثبت الكامل
- `نظام إدارة الحضور-Portable-1.0.0.exe` - النسخة المحمولة
- مجلد `win-unpacked` - الملفات غير المضغوطة

## ملاحظات مهمة

1. **متطلبات التشغيل على الأجهزة الأخرى:**
   - Windows 10/11 (64-bit)
   - لا يحتاج Node.js أو أي تبعيات
   - يحتاج كود تفعيل فريد لكل جهاز

2. **حجم التطبيق:**
   - حوالي 150-200 ميجابايت (يشمل Node.js و Chromium)

3. **نظام الحماية:**
   - كل جهاز له معرف فريد (Hardware ID)
   - كود التفعيل مرتبط بالجهاز ولا يمكن نقله

4. **التوزيع:**
   - وزع ملف Setup.exe للمستخدمين
   - احصل على Device ID من كل مستخدم
   - أنشئ لهم أكواد تفعيل من LicenseKeyGenerator

## استكشاف الأخطاء

### خطأ في البناء:
```cmd
npm install
npm run build
```

### تنظيف البناء السابق:
احذف مجلد `D:\AttendanceSystem-Build` ثم أعد البناء

### اختبار التطبيق قبل البناء:
```cmd
npm run electron
```
