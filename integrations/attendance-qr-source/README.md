## 🔧 نظام الترخيص (الميزة الجديدة)

### كيف يعمل:
1. **التشفير**: يتم تشفير مفتاح الترخيص باستخدام AES-256-CBC
2. **التحقق**: عند كل تشغيل، يتم التحقق من صحة المفتاح
3. **الحماية**: يتم فحص Hardware ID وعدم التلاعب بالوقت
4. **الحرق**: بعد الاستخدام الأول، لا يمكن استخدام المفتاح مرة أخرى
5. **التنبيه**: تنبيهات ذكية عند اقتراب انتهاء الترخيص

### مولد المفاتيح:
- موجود في: `LicenseKeyGenerator/index.html`
- يعمل أوفلاين بالكامل (لا يحتاج إنترنت)
- سهل الاستخدام: (نسخ معرف الجهاز → اختر مدة → توليد)

### أسعار الترخيص المقترحة:
```
7 أيام   → 50 ريال
30 يوم   → 150 ريال
90 يوم   → 400 ريال
180 يوم  → 700 ريال
365 يوم  → 1200 ريال
3 سنوات  → 2500 ريال
```

## 🔐 الأمان

النظام يستخدم **عدة طبقات حماية**:

### مستوى 1: التشفير
- AES-256-CBC (معيار عسكري)
- مفتاح 32 بايت
- IV عشوائي لكل مفتاح

### مستوى 2: الربط بالجهاز
- Hardware ID فريد لكل جهاز
- المفتاح يحتوي على معرف الجهاز
- لا يعمل على أي جهاز آخر

### مستوى 3: الاستخدام الواحد
- SHA-256 hashing للمفاتيح
- قائمة محروقة في موقعين
- منع إعادة الاستخدام تماماً

### مستوى 4: حماية الوقت
- Time Ratchet system
- كشف تعديل ساعة النظام
- رفض الترخيص عند الكشف

### مستوى 5: التخزين المزدوج
- electron-store
- ملفات احتياطية مخفية
- حماية ضد تلاعب البيانات
