## 📞 في حالة المشاكل

### المشكلة: التطبيق لا يفتح بعد التحديث

✅ **الحل:**
```powershell
1. افتح Task Manager (Ctrl+Shift+Esc)
2. أنهِ جميع عمليات "نظام إدارة الحضور"
3. أعد تشغيل الكمبيوتر
4. افتح التطبيق مرة أخرى
```

### المشكلة: لا أرى بياناتي

✅ **الحل:**
```
1. تحقق من وجود الملفات في:
   %APPDATA%\student-attendance-center\databases\

2. إذا كانت موجودة، المشكلة في العرض فقط (أعد التشغيل)

3. إذا لم تكن موجودة، استرجع من النسخة الاحتياطية:
   Documents\AttendanceBackups\
```

### المشكلة: أريد الرجوع للنسخة 1.0.0

✅ **الحل:**
```
1. ألغِ تثبيت 1.0.1 (Settings > Add/Remove Programs)
2. ثبّت 1.0.0 من جديد
3. بياناتك ستبقى كما هي ✅
```
